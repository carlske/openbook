# Open Book

# ⚛️📦 Project with ⚡ Vite & 🚀 pnpm

This is a ⚛️ React project created with ⚡ Vite and managed with 🚀 pnpm. The 📱 application consumes a 📚 public book API in 🇬🇧 English to display 🔍 relevant information about various 📖 titles.

## 🛠️ Technologies Used

- [⚡ Vite](https://vitejs.dev/) - ⚡ Fast development environment
- [⚛️ React](https://react.dev/) - 🖥️ Library for 🖌️ user interfaces
- [🚀 pnpm](https://pnpm.io/) - ⚡ Fast & 🏎️ efficient 📦 package manager

## ⚙️ Installation

Make sure you have `🚀 pnpm` installed on your 💻 system. If not, install it with:

```sh
npm install -g pnpm
```

Then, 🏗️ clone the 📂 repository & install dependencies:

```sh
git clone https://github.com/your-username/your-repository.git
cd your-repository
pnpm install
```

## 🏗️ Development Server

To start the 🚀 dev server, run:

```sh
pnpm dev
```

The 🌍 application will be available at `http://localhost:5173/`.

## 🏭 Build for Production

To generate the 📦 production build, use:

```sh
pnpm build
```

The 📂 files will be generated in the `dist/` folder.

## 📚 Book API Usage

This 📱 app consumes a 📚 public book API in 🇬🇧 English. Be sure to 📖 review the 📜 API documentation to understand its 🔗 endpoints and 🛠️ parameters.

Example request to the API:

```js
fetch('https://api.example.com/books?language=en')
  .then(response => response.json())
  .then(data => console.log(data));
```
